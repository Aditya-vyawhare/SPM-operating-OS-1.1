# Skill: GMV Growth Operating Framework

Use this skill when asked to improve GMV, prioritize growth initiatives, or build a quarter plan.

## Inputs required

- Current GMV by mode (flight/hotel/bus/train)
- Funnel conversion by stage
- AOV by segment
- Repeat/retention data
- Leakage baseline by category

## Step-by-step workflow

## 1) Diagnose growth equation

Break GMV into:

- traffic x conversion x AOV x repeat
- subtract leakage to estimate realized GMV

## 2) Identify highest-leverage constraints

Rank opportunities by:

- expected GMV uplift
- implementation effort
- time-to-impact
- confidence level

## 3) Build initiative portfolio

Cover four levers:

- **Funnel optimization:** reduce search-to-book drop-off
- **Pricing quality:** improve fare consistency, reduce checkout surprises
- **Upsell and ancillaries:** increase attach rate and package penetration
- **Retention:** increase repeat at traveler and account levels

## 4) Set measurable targets

For each initiative define:

- baseline metric
- target delta
- date to impact
- owner

## 5) Install operating cadence

- Weekly growth review: conversion, AOV, attach, repeat
- Weekly leakage review: recovered value, unresolved exceptions
- Monthly strategy checkpoint: ROI reallocation across bets

## Output format

Return:

1. Top 3 GMV opportunities with estimated upside
2. Effort vs ROI matrix
3. 30-60-90 execution plan
4. Risks and guardrails
